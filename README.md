# Heading Trainer

Offline/PWA webová aplikace pro procvičování sčítání a odčítání headingů a headingů jednotlivých částí pravého či levého letištního okruhu.

Headingy používají rozsah 001–360 a vždy se zobrazují i zadávají jako trojmístné hodnoty. U dráhy se nominální heading počítá jako číslo dráhy × 10; dráha 36 odpovídá 360°.

Po publikování přes HTTPS lze aplikaci přidat na plochu telefonu. Pokrok se ukládá pouze lokálně v prohlížeči.
